# Event Platform
Event management system made for practice with ASP.NET and React.

## Execution
### Requirements
- Node.js
- .NET 8 SDK
- Docker
- PostgreSQL

### URLs
- Backend: http://localhost:5000/swagger
- Frontend: http://localhost:3000

### Run Build
```bash
docker compose up
```

### Run Dev
```bash
docker compose up -f docker-compose.yml -f docker-compose.dev.yml
```
